# Dark/Light Mode Toggle 

A responsive Dark/Light theme toggle for websites with smooth transitions, creative icons, and persistent user preference using `localStorage`.

## Features

* Toggle between **Dark** and **Light** themes.
* Smooth **color transitions** for all elements.
* **Sun & Moon icons** in the toggle for a creative look.
* **Persistent theme preference** using `localStorage`.
* Fully **responsive** and mobile-friendly design.
* Easy to integrate into any project.

## Usage

* Click the toggle switch to switch between **dark** and **light** mode.
* Your preference will be **saved automatically** and applied on next visit.

## Files

* `index.html` – The main HTML file.
* `style.css` – Styling including transitions, toggle design, and dark/light mode colors.
* `script.js` – JavaScript handling the toggle and saving preference in `localStorage`.
